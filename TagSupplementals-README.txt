TagSupplementals - Supplemental Features for handling "tags".

= Overview =

TagSupplementals plugin provides several block and function tags for
handling "tag" extensively.

= Installation =

 * Download and extract TagSupplementals-<version>.zip file.

 * Upload or copy the contents of "plugins" directory into your
   "plugins" directory.

 * After proper installation, you will find "TagSupplemental" plugin
   listed on the "System Plugin Settings" screen.
